# Hospital Management System (Java Console Application)

This is a **menu-driven Hospital Management System** developed in Java as my **Class 10 Computer Science Board Project**.  
It simulates the basic working of a hospital by managing **doctors, patients, laboratories, facilities, and staff records** using a console-based interface.

---

## Features

- Add and view **Doctor records**
- Add and view **Patient records**
- Manage **Laboratory facilities and costs**
- View **Hospital facilities**
- Manage **Staff (Nurses, Workers, Security)**
- Uses a **menu-driven interface**
- Displays **date and time** using Java Calendar class

---

## Technology Used

- **Language:** Java  
- **Concepts:**  
  - Classes & Objects  
  - Arrays  
  - Scanner input  
  - Conditional statements  
  - Loops  
  - Menu-driven programming  
- **Platform:** Console / Command Line

---

## How the System Works

- The program initializes some **default doctors, patients, labs, facilities, and staff**.
- User is shown a **Main Menu** with options:
  1. Doctors  
  2. Patients  
  3. Laboratories  
  4. Facilities  
  5. Staff  
- Each section allows:
  - Adding new records
  - Viewing existing records
- All data is handled using **arrays of objects**.
